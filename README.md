# Poznań Rent Price Prediction

Machine learning pipeline for predicting apartment rental prices in Poznań using incomplete tabular and textual real-estate listing data.

## Project overview

The goal of this project is to build an end-to-end machine learning solution for predicting apartment rental prices based on real-estate listing data.

The dataset contains structured numerical and categorical features, missing values, and textual descriptions of rental offers. The project focuses on building a robust modelling pipeline that combines tabular and text-based features, evaluates multiple models using cross-validation, and produces final predictions using a stacked ensemble.

## Methods

The solution includes:

- data loading and validation,
- missing value handling,
- preprocessing of numerical and categorical variables,
- feature engineering for tabular real-estate attributes,
- text feature extraction using TF-IDF,
- dimensionality reduction of text features with TruncatedSVD,
- model training with LightGBM, XGBoost and Ridge Regression,
- out-of-fold cross-validation,
- final model stacking using Polynomial Ridge Regression.

## Results

| Model | Validation strategy | OOF MSE |
|---|---:|---:|
| LightGBM tabular | 5-fold CV | 90,326 |
| XGBoost tabular | 5-fold CV | 89,817 |
| TF-IDF + Ridge | 5-fold CV | 155,122 |
| LightGBM + SVD text | 5-fold CV | 82,691 |
| Polynomial Ridge Stack | 5-fold CV | 75,757 |

The final stacked model achieved the best validation performance with an out-of-fold MSE of approximately **75,757**.

The final prediction file is saved as:

```text
outputs/submission_polystack.csv
```

## Data and output files

The notebook expects the following input files in the `data/` directory:

- `pzn-rent-train.csv` — training dataset with apartment rental listings and target prices,
- `pzn-rent-test.csv` — test dataset used for generating predictions,
- `pzn-sample-sub.csv` — sample submission file with the required output format.

The final output file is saved in the `outputs/` directory:

- `submission_polystack.csv` — final predictions generated by the stacked model.

## How to run

Clone the repository:

```bash
git clone https://github.com/Kiedro12/poznan-rent-price-prediction.git
cd poznan-rent-price-prediction
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Open the notebook:

```bash
jupyter notebook notebooks/poznan_rent_price_prediction.ipynb
```

The notebook uses relative paths, so it can be run from the repository root or from inside the `notebooks/` directory.
